# 亚马逊搜索词加否 Agent

一个**纯前端单页工具**，上传亚马逊广告搜索词报告后，自动判断哪些搜索词建议加否，并导出加否清单。

## 特点

- **本地运行，数据不外传**：所有解析和分析都在你的浏览器里完成，报告不会上传到任何服务器。
- **支持 xlsx / xls / csv**：直接上传亚马逊后台导出的搜索词报告。
- **三条加否规则（可在页面开关 + 调阈值）**：
  1. **点击多不出单**：点击数 ≥ 阈值 且 订单 = 0
  2. **转化率过低**：有订单但转化率 < 阈值（且点击量足够，样本可信）
  3. **无关词黑名单**：搜索词包含你设定的竞品词/无关词
- **智能识别列名**：自动匹配中英文表头（搜索词 / Search Term、点击 / Clicks 等）。
- **一键导出**：导出建议加否的搜索词 CSV，整理后回亚马逊后台批量添加否定关键词。

## 使用方法

1. 用浏览器打开 `index.html`（双击即可）。
2. 在「规则与阈值」里按你的标准调整数值、填入无关词黑名单。
3. 上传搜索词报告。
4. 查看「建议加否」结果，点击表头排序，点「导出加否清单」。

> 提示：阈值改动后结果会自动重算。规则①「点击多不出单」是核心，无关词黑名单需要你自己维护竞品词/不相关词列表。

## 说明

- xlsx 解析使用 SheetJS（通过 CDN 加载）。若处于离线/受限网络导致 xlsx 无法解析，请改用 CSV 文件。
- 「无效花费(估)」= 所有 0 订单的加否词花费之和，帮你直观看到每天浪费了多少广告费。
